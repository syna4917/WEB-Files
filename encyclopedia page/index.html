Intro to Pure Barre

Page Format:

  1. Class Structure
  2. Class Types

Pure Barre is a fitness class rooted in low impact excersize that fatigues the muscles.

Class Structure

Each Classic class follows the same format:

  1. Warm Up
  2. Weight Work
  3. Thigh Work
  4. Seat Work
  5. Core Work
  6. Final Seat Blast

Each section of class is followed by a stretch of that muscle group.

Class Types

  • Classic:The most common class type and the basis for all Pure Barre technique. Equipment includes a ball, a double tube, and a light and heavy set of weights
  • Empower:A cardio based technique that keeps the heartrate high with much shorter breaks. Equipment includes a platform, wrist/ankle weights, and a light set of weights.
  • Reform:Resistance technique that makes use of stagnant holds for strengthening. Equipment includes 2 slides and 4 resistance tubes.
